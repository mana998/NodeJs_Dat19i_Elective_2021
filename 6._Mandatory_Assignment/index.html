Getting Started

What is Node.js?

V8 javascript engine taken from browser to run javascript without browser - can execute everywhere

    Can create:
  • Websites
  • Desktop applications
  • Scripts
  • Command line interfaces

Creating new project

Create package.json file. 1. Create empty json and run "npm install express" or 2. Create key value pair with key "dependencies" and value object: key - name of dependency value - version repeat for all modules you want to add

Setting up the server

1. Instantiate express object 2. Set up http endpoints 3. Start the server on a port

Running the project

0. node - opens REPL prompt (read, evaluate, print, loop) - for executing commands 1. node app.js - needs to restart after each change 2. nodemon[.cmd] app.js - automatically restarts when change occurs
