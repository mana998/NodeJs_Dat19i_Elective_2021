jQuery

What is it?

Unified lightweight JavaScript library for manipulating the DOM elements across different browsers. Furthermore it contains Ajax. It provides easy functions for accessing, modifying, traversing different elements.

Ajax

Asynchronous javascript and XML. Used for fetching resources without refreshing the whole page by using callbacks.

How to get it?

It can be easily served from Content Delivery Network. For example from google's CDN. 

    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>

Content Delivery Network

Group of servers that provide static files. Improves speed of serving static content

How to use it?

We can select elements the same way as in CSS by using a selector.

  • Tag 
    ${"body"}
  • Class 
    ${".className"}
  • Id 
    ${"#id"}
Some methods examples to use with selectors:
  • Get the text: 
    ${"#id"}.text()
  • Modify text: 
    ${"#id"}.text("New text")
  • Append at the end 
    ${"#id"}.append("This is at the end")
  • Prepend at the start 
    const title = `<h1>This is title</h1>`
${"#id"}.prepend(title)
  • Update CSS 
    ${"#id"}.css("background-color", "red")
  • Add event listener 
    ${document}.on("click", callback function)
  • Remove element
    ${"#id"}.remove()
  • Remove element from element
    ${"#id elementToRemove"}.remove()
