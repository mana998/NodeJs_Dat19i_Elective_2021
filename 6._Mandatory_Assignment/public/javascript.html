Const vs Let

Const - cannot change value
Let - can change value

Callback Functions

Function passed into another function as an argument. It can be then executed right away or after some other commands 

function sayHiLater(callbackFunction) {
    console.log("Doing something imporant");
    console.log("Loading");
    console.log("Almost done");
    callbackFunction();
}
//pass arrow function                            
sayHiLater(() => {console.log("Hi")});
/*
    output:
    Doing something important
    Loading
    Almost done
    Hi
*/

//save arrow function into const
const sayHi = () => {
    console.log("Hello!");
}
//pass const
sayHiLater(sayHi);
/*
    output:
    Doing something important
    Loading
    Almost done
    Hello!
*/

Loops

for (item of array) - loops through all items of array

const fruits = ["apples", "banana", "blueberries"];
for (fruit of fruits) console.log(fruit);
for (property in object) - loops through all keys/properties of an object
let member = {
    username: "Lia",
    candies: 5
};
for (property in member) { console.log("property ", property, " value ", member[property]);}
array.forEach((element, index, array) => {}) Unlike in "for of" loop, we can specify additional index and array parameter
fruits.forEach((fruit, index, array) => console.log(fruit, "index: ", index, "full array: ", array));

Functional loops examples:

array.map((element) => element/function) - function runs for each value and returns each value in a new array. We can either specify what modifications should happen with the value or simply return the whole or part of the original object.

const newFruit = fruits.map(fruit => fruit); //returns fruit
array.reduce((accumulator, element) => function); - We have 2 parameters, accumulator and element. We can specify existing variable for an accumulator or specify its starting value as a 2nd parameter. After each run of the function, final value is stored in the accumulator and then the loop is run for another element. At the end we end up with a single value returned.
const members = [
    {
        username: "Lia",
        candies: 10
    }, 
    {
        username: "Leon",
        candies: 3
    }, 
];
const sum = members.reduce((total, member) => total + member.candies, 0);
array.filter(element => function); - In the function, we specify a condition that must apply. If the return value is true, elemet is added into the new array, else it is discarded.
const aboveFive = members.filter((member) => member.candies > 5);
